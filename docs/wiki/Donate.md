# Support LSPV

LSPV is built and maintained in spare time with no corporate backing, no VC funding, and no plan to monetize. There are no ads, no premium tiers, and no data collection.

If LSPV is useful to you — for keeping your passwords safe, for learning how serious crypto engineering looks in Rust, or for having a manager you actually control — please consider supporting the project.

## What donations go toward

- Development time (this competes with a day job)
- Code signing certificates for Windows installer (prevents "unknown publisher" warnings)
- macOS notarization fees (required for app to run without Gatekeeper bypass)
- Security auditing tools and services

## Other ways to support

- **Star the repo** — helps LSPV get discovered on GitHub search and trending
- **Open issues** — bug reports and feature requests help prioritize work
- **Contribute code** — see [CONTRIBUTING.md](../../CONTRIBUTING.md)
- **Spread the word** — tell developers who care about privacy that this exists

## What LSPV will never do

- Introduce a "pro" version behind a paywall
- Add telemetry or usage tracking to unlock funding
- Sell or share any user data (there is none — no server exists)
- Add cloud sync unless it is end-to-end encrypted and fully optional

Thank you.
